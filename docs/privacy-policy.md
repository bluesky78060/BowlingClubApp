# 개인정보처리방침

## 볼링클럽 앱

**최종 수정일**: 2026-02-09

### 1. 수집하는 개인정보

볼링클럽 앱은 다음 정보를 기기 내에서만 저장합니다:
- 회원 이름, 연락처, 성별
- 볼링 점수 기록
- 프로필 사진 (선택)

### 2. 개인정보의 이용 목적

수집된 정보는 다음 목적으로만 사용됩니다:
- 동호회 회원 관리
- 볼링 점수 기록 및 통계 분석
- 순위 산출 및 결과 공유

### 3. 개인정보의 보관 및 파기

- 모든 데이터는 사용자의 기기에만 저장됩니다
- 외부 서버로 전송되지 않습니다
- 앱 삭제 시 모든 데이터가 자동으로 삭제됩니다
- 백업 파일은 사용자가 직접 관리합니다

### 4. 외부 서비스

- **Naver CLOVA OCR**: 점수판 이미지 인식 시에만 사용
  - 촬영한 이미지가 OCR API로 전송됩니다
  - 인식 결과 수신 후 이미지는 서버에서 즉시 삭제됩니다

### 5. 카메라 권한

- 점수판 OCR 촬영 목적으로만 사용됩니다
- 촬영된 이미지는 기기 내 캐시에만 임시 저장됩니다
